# Maze Runner

A 2D top-down roguelite built in Unity where players explore procedurally generated mazes, defeat enemies, collect materials, place checkpoints, and escape through the exit portal. Every run features a newly generated maze, making each playthrough unique.

## Features

- Procedurally generated mazes
- Top-down player movement
- Mouse-based shooting
- Enemy combat
- Resource collection
- Checkpoint placement system
- Portal-based level progression
- Endless replayability through random generation

## Controls

| Action | Key |
|--------|-----|
| Move | W / A / S / D |
| Shoot | Left Mouse Button |
| Place Checkpoint | C |

> **Tip:** Collect all three materials before placing a checkpoint.

## Gameplay

1. Spawn in a randomly generated maze.
2. Explore while avoiding or defeating enemies.
3. Collect all three materials.
4. Press **C** to place a checkpoint.
5. Find the exit portal.
6. Enter the portal to generate a new maze.
7. Repeat and survive as long as possible.

## Technologies Used

- Unity
- C#
- Unity Tilemap System
- Procedural Maze Generation

## AI Usage

AI was used to:
- Debug Unity C# scripts
- Generate development images

AI is **not** part of the gameplay. The focus of this prototype was to build and validate a fun core gameplay loop.
